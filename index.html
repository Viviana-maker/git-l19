Working Directory

Es el área de trabajo, donde rara vez utilizaremos la línea de comandos y especialmente donde estaremos realizando las modificaciones dentro de nuestro proyecto, como la creación de archivos, la modificación de los mismos e incluso su eliminación.

  • dir || ls Muestra el contenido de la carpeta donde nos encontramos actualmente.
  • cd <path> Cambia el directorio actual a la ruta especificada
  • mkdir <name Crea una carpeta con el nombre indicado
  • git init Inicializa un repositorio de Git en el directorio actual
  • rm -rf .git Elimina el repositorio de git, el parámetr -rf indica que debe eliminar la carpeta de manera forzada
